1.Primero en la carpeta que tenga clonada haga el comando git pull 
2.Luego git add .
3.Despues agregue su commit  git commit  -m "Escriba lo que hizo"
4.Haga un git push
5.Despues siempre que alguien haga una modificacion utilizar git pull para traer todas las actualizaciones
6.No trabajar en las mismas escenas a el mismo tiempo crearia un conflicto
