INSSTRUCCIONES PARA GIT 

1. Primero en la carpeta que tenga clonada haga el comando git pull para traer todas las actualizaciones

2. Luego git add . (en algunos casos no es necesario)

4. Despues agregue su commit  git commit  -m "Escriba lo que hizo"  (para subir lo que hizo)

5. Haga un git push (asi lo sube a el git y verifica en la pagina que si se haya actualizado)

6. Despues siempre que alguien haga una modificacion utilizar git pull para traer todas las actualizaciones

7. No trabajar en las mismas escenas a el mismo tiempo crearia un conflicto
